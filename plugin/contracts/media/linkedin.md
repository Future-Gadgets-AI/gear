# LinkedIn

Renders a post's canonical spec as a LinkedIn-native post — a derivation of the spec, never an independent rewrite (Constitution Principle 1).

## Frontmatter

Inherits `content-spec.md` in full — see that file. Consumes: `tl_dr`, one pull-quote.

## Rules

1. The hook lands in the first ≤210 characters (mobile truncates around 120–140 chars, so the strongest claim goes first).
2. NO URL in body — a body URL kills reach on the platform's algorithm.
3. The final line is literally "Link in first comment:" followed by `canonical_url`.
4. ≤3 hashtags, or none.
5. Content is derivation-only, built from `tl_dr` and one pull-quote — never a rewrite of the full article.

## Structure notes

- Order: hook → value → CTA.
- Short paragraphs, generous line breaks.
- Frame for save-worthy value, not just a read-and-scroll.
